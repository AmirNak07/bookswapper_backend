# Bookswapper backend

Stack/Libs used:
- Bcrypt
- Fiber
- Gorm
- PostgreSQL

Configuration via environment variables (```.env``` file for docker):
```
POSTGRES_HOST=docker
POSTGRES_PORT=5432
POSTGRES_USER=postgres
POSTGRES_PASSWORD=postgres
POSTGRES_DB=bookswapper
```
